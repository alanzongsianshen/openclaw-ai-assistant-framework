# Skill深度学习机制

## 🎯 目标

不只是安装skill，而是深度学习skill中的知识

## 📚 学习流程

### 1. 安装skill
- 从ClawHub安装skill

### 2. 阅读skill内容
- 查看skill目录结构
- 阅读SKILL.md（如果存在）
- 阅读主要代码文件

### 3. 提取知识点
- 核心方法
- 最佳实践
- 设计模式
- 使用技巧

### 4. 整合知识
- 关联已有知识
- 发现共同模式
- 创造新方法

### 5. 记录到知识库
- 保存到 `data/skill-knowledge-base.json`

## 📊 知识库结构

```json
{
  "skills": [
    {
      "name": "skill-name",
      "category": "category",
      "knowledge_points": [
        {
          "point": "知识点1",
          "description": "描述",
          "usage": "使用场景"
        }
      ],
      "best_practices": [
        "最佳实践1",
        "最佳实践2"
      ],
      "patterns": [
        "设计模式1"
      ],
      "learned_at": "2026-02-28 12:20"
    }
  ]
}
```

## 🔄 自动化

### 每次安装新skill后
1. 自动读取skill内容
2. 提取知识点
3. 保存到知识库
4. 生成学习报告

### 每周
1. 分析知识库
2. 发现知识关联
3. 创造新方法
4. 更新能力
